# Sistema de Banca Digital – Transferencias entre Cuentas

## Descripción

Proyecto de QA Manual enfocado en la validación del módulo de transferencias entre cuentas de una aplicación bancaria.

## Objetivo

Garantizar que las transferencias se ejecuten correctamente y cumplan con las reglas de negocio establecidas.

## Artefactos del Proyecto

- Requisitos Funcionales
- Matriz de Trazabilidad
- Casos de Prueba
- Reporte de Defectos
- Test Summary Report

## Herramientas Utilizadas

- Excel
- QA Manual

## Técnicas Aplicadas

- Partición de Equivalencia
- Análisis de Valores Límite
- Pruebas Funcionales
- Pruebas Negativas

## Resultados

Se diseñaron 25 casos de prueba y se identificaron defectos que permitieron detectar oportunidades de mejora antes de una posible liberación a producción.
